This example demonstrates the use of Rapid SCADA JavaScript API.
It's compatible with the HelloWorld project.

How to deploy:
1. Copy the custom directory into C:\SCADA\ScadaWeb\
2. Open the project in Administrator and add a new record in the Interface table:
+--------------------------+-------------+--------------------+
| Path                     | View Type   | Title              |
+--------------------------+-------------+--------------------+
| ../../custom/MyPage.html | WebPageView | HelloWorld\My Page |
+--------------------------+-------------+--------------------+
3. Upload the configuration.
