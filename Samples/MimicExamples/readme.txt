Copy the mimics and faceplates to the Views\Mimics\ directory of the HelloWorld project.
Don't forget to add rows corresponding to *.mim files to the Views table.

CommandBox.fp        A faceplate for sending a command whose value is entered using a text field.
Damper.fp            A faceplate demonstrating how to work with SVG using scripts.
ExtraCompTest.mim    A mimic diagram demonstrating the use of extra components.
MyFaceplate.fp       A simple faceplate used in MyMimic.mim
MyMimic.mim          Basic mimic diagram.
ScriptTest.mim       Demonstrates how components interact using scripts.
SunComponent.fp      An example of a faceplate displaying the sun.
